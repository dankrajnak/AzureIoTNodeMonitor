    public addRunInBackgroundListener(eventName: string, fileName: string) source

    Listens for an eventName device method. Once received, create a new instance of node running in a seperate process and run fileName. All output from that process will be sent to the IoTHub as an event.

    Params:

    NameTypeAttributeDescription
    eventName string

    Name of event to listen to from the IotHub
    fileName string

    Relative path to the javascript file

    public addSameThreadMethodListener(eventName: string, method: function) source

    Listens for an eventName device method. Once received, runs the provided method in the same thread as the IoT Monitor. This is meant for short methods and avoids the creation of a separate process. Consider, though, that while the method is running, IoTMonitor will be unresponsive.

    Params:

    NameTypeAttributeDescription
    eventName string

    name of the device method to listen for
    method function

    method to run on reciept of the eventName

    Run a method on the same thread as the IoTMonitor. Meant for simple, synchronous methods which quickly terminate. Avoids creating another Node instance.
